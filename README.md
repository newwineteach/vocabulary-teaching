# MTC Online 師資班 - 詞彙教學互動課程 🎓

> 第47期師資班詞彙教學講義 - 互動式網頁版本

[![部署狀態](https://img.shields.io/badge/status-已部署-success)](https://newwineteach.github.io/mtc-vocabulary-teaching/)
[![授權](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## 📚 課程簡介

這是一個為華語教師設計的互動式教學平台，將詞彙教學理論轉化為6個沉浸式的學習單元。通過普普藝術風格的視覺設計和現代Web技術，讓教學法的學習變得生動有趣！

---

## 🎮 六大互動單元

### 1️⃣ 你的大腦喜歡怎麼學？
**VARK學習風格測試**
- 🎯 了解學生的學習偏好
- 📊 視覺型、聽覺型、讀寫型、動覺型
- 🎨 普普風格互動測驗

### 2️⃣ 成語卡牌大亂鬥
**多人協作遊戲**
- 🎴 拖曳式卡牌互動
- 👥 支援4個隊伍
- 🎯 成語教學實戰演練

### 3️⃣ 初級詞彙教學小結
**Flashcards翻卡練習**
- 📇 12張精選複習卡
- 🔄 洗牌和重置功能
- 💡 核心教學原則總結

### 4️⃣ 各級學習需求與表現
**詞彙分類遊戲**
- 📊 初級、中級、高級分類
- 🎯 拖曳或點擊操作
- ✅ 即時正誤反饋

### 5️⃣ 近義詞波普大作戰
**辨析挑戰測驗**
- 🎨 7大類近義詞辨析
- 💥 普普藝術風格
- 📖 詳細解析說明

### 6️⃣ 詞彙教學策略總測驗
**Matrix風格終極測驗**
- 🌐 駭客帝國主題
- 🏆 計分排行榜
- 📈 10題綜合評估

## 🚀 快速開始

### 線上訪問
直接訪問：[https://newwineteach.github.io/mtc-vocabulary-teaching/](https://newwineteach.github.io/mtc-vocabulary-teaching/)

### 本地運行
```bash
# 克隆倉庫
git clone https://github.com/newwineteach/mtc-vocabulary-teaching.git

# 進入目錄
cd mtc-vocabulary-teaching

# 使用任何本地服務器即可運行，例如：
# Python 3
python -m http.server 8000

# 或直接在瀏覽器中打開 index.html
```

## 🛠️ 技術架構

### 前端技術棧
- **HTML5** - 語義化結構
- **Tailwind CSS** - 實用優先的CSS框架
- **React 18** (CDN版本) - 聲明式UI組件
- **Babel Standalone** - 瀏覽器端JSX編譯

### 設計特色
- ✨ **普普藝術風格** - 鮮豔色彩、粗黑邊框、陰影效果
- 📱 **響應式設計** - 支援桌面、平板、手機
- 🎨 **多種視覺主題** - 每個單元獨特的配色方案
- ⚡ **零編譯部署** - 直接發布到GitHub Pages

### 無需後端
- 🚫 無需Node.js編譯
- 🚫 無需Firebase（純前端版本）
- 🚫 無需服務器
- ✅ 完全靜態、即開即用

## 📖 教學使用指南

### 適用場景
- 🎓 **師資培訓課程** - 作為課堂互動工具
- 📚 **自主學習** - 教師課後複習鞏固
- 💻 **翻轉教學** - 學生課前預習
- 🎮 **教學演示** - 詞彙教學法展示

### 建議流程
1. **課前預習** → Lesson 1-3（了解基礎）
2. **課堂討論** → Lesson 4-5（深入辨析）
3. **課後鞏固** → Lesson 6（總結評估）
4. **實戰演練** → Lesson 2（遊戲化練習）
